Funciones en JavaScript

Las funciones son bloques de código reutilizables diseñados para realizar tareas específicas. Son fundamentales en JavaScript porque:

01

  • Organizan el código
  • Evitan repetición
  • Dan contexto semántico a las acciones

02

  • Permiten escribir funciones con menos código.
  • En arrow functions, this es el mismo que el de su entorno.
  • Cuando la arrow function tiene una sola línea, no se necesita return.

Callbacks en JavaScript

Es una función que se pasa como argumento a otra función y que se ejecuta después de que ocurra algún evento. Son fundamentales en JavaScript porque permiten manejar operaciones asíncronas. Como funciona:

03

  • Se pasa una función como argumento a otra función.
  • La función principal realiza alguna tarea.
  • Cuando termina, se ejecuta el callback.

Promesas en JavaScript

Las promesas son objetos que representan la eventual finalización (éxito o fracaso) de una operación asíncrona. Usarlas porque:

04

  • Evitan callbacks anidados permitiendo un código
    más plano y legible.
  • Usan ".catch()" para centralizar el manejo de errores
  • Permiten ejecutar operaciones encadenadas con
    ".then()".
